Exporting to OuDia Format

You can export the current diagram to .oud in case if you need to further edit the diagram in OuDia or OuDiaSecond.

File encoding

By default, .oud files would use Shift-JIS. This is fine if you are only using English or Japanese. This is to ensure that both OuDia and OuDiaSecond could understand the file content.

UTF-8 and .oud2 export are currently unsupported.
